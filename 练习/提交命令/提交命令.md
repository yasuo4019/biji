
## 日常更新

进入项目目录：

```bash
cd C:\Users\why\Desktop\biji
```

先拉最新内容：

```bash
git pull
```

查看改动：

```bash
git status
```

提交并上传：

```bash
git add .
git commit -m "本次更新说明"
git push
```


---

## 说明

- `git pull`：先拉 GitHub 最新内容
- `git status`：看哪些文件变了
- `git add .`：把改动加入提交
- `git commit -m "..."`：生成一次提交
- `git push`：上传到 GitHub
