# 1. 基础查询：过滤 + 排序 + 分页

### 1.1 查所有有效客户（status=1），按创建时间倒序

```sql
select customer_no, full_name, phone, status, created_at
from yewu.customer
where status = 1
order by created_at desc
limit 20;
```

### 1.2 模糊查询：姓“王”的客户

```sql

from yewu.customer

```

### 1.3 查余额最高的 5 个账户

```sql

from yewu.account
```

# 2. JOIN：把“业务对象”拼起来

### 2.1 客户-账户（一个客户可能多个账户）

```sql
select
  c.customer_no, c.full_name,
  a.account_no, a.currency, a.account_type, a.balance, a.status as account_status
from yewu.customer c
join yewu.account a on a.customer_id = c.customer_id
order by c.customer_no, a.account_no;
```

### 2.2 客户-账户-交易流水（最常见业务查询）

```sql
select
  c.customer_no, c.full_name,
  a.account_no,
  t.txn_id, t.request_id, t.direction, t.amount, t.txn_type, t.status, t.created_at
from yewu.customer c
join yewu.account a on a.customer_id = c.customer_id
join yewu.txn t on t.account_id = a.account_id
order by t.created_at desc
limit 50;
```

# 3. 聚合统计：GROUP BY / HAVING

### 3.1 每个客户有多少账户

```sql
select
  c.customer_no, c.full_name,
  count(*) as account_cnt
from yewu.customer c
join yewu.account a on a.customer_id = c.customer_id
group by c.customer_no, c.full_name
order by account_cnt desc, c.customer_no;
```

### 3.2 每个账户的交易笔数、入账总额、出账总额

```sql
select
  a.account_no,
  count(*) as txn_cnt,
  sum(case when t.direction = 1  then t.amount else 0 end) as in_sum,
  sum(case when t.direction = -1 then t.amount else 0 end) as out_sum
from yewu.account a
join yewu.txn t on t.account_id = a.account_id
group by a.account_no
order by txn_cnt desc, a.account_no;
```

### 3.3 找“交易笔数 >= 3”的账户（HAVING）

```sql
select
  a.account_no,
  count(*) as txn_cnt
from yewu.account a
join yewu.txn t on t.account_id = a.account_id
group by a.account_no
having count(*) >= 3
order by txn_cnt desc;
```

# 4. 窗口函数入门：Top N、累计、排序编号

### 4.1 每个账户最近 1 笔交易（Top 1 per group）

```sql
select *
from (
  select
    a.account_no,
    t.*,
    row_number() over (partition by t.account_id order by t.created_at desc) as rn
  from yewu.txn t
  join yewu.account a on a.account_id = t.account_id
) x
where x.rn = 1
order by x.created_at desc;
```

### 4.2 每个账户按时间的“累计净额”（入-出）

```sql
select
  a.account_no,
  t.created_at,
  t.direction,
  t.amount,
  (t.direction * t.amount) as signed_amount,
  sum(t.direction * t.amount) over (
    partition by t.account_id
    order by t.created_at
    rows between unbounded preceding and current row
  ) as running_net
from yewu.txn t
join yewu.account a on a.account_id = t.account_id
order by a.account_no, t.created_at;
```
