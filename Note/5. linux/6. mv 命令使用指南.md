
`mv` 命令用于在 Linux/Unix 系统中移动文件或目录，也可用于文件或目录的重命名。它可以将一个文件或目录从一个位置移动到另一个位置，同时进行文件或目录的重命名操作。

## 命令格式
```bash
mv [选项] 源文件 目标文件
```

## 常用选项

- `-i` 或 `--interactive`：如果目标文件已存在，提示是否覆盖。
- `-v` 或 `--verbose`：显示详细的移动过程信息。

## 示例

1. **移动文件或目录**：
   ```bash
mv source_file destination_file
   ```
   例如：
   ```bash
mv L13.mt.vcf ./SXB/test.mt.vcf
   ```
   这将移动名为 "L13.mt.vcf" 的源文件，并将其重命名为 "test.mt.vcf" 并放置在目标路径中 (`./SXB`)。

2. **重命名文件或目录**：
   ```bash
mv old_name new_name
   ```
   例如：
   ```bash
mv ./SXB/test.mt.vcf ./SXB/test.vcf
   ```
   这将名为 "test.mt.vcf" 的文件重命名为 "test.vcf"。

   ```bash
mv SXB/Linux/Test SXB/Linux/Test1
   ```
   这将名为 "Test" 的目录重命名为 "Test1"。
