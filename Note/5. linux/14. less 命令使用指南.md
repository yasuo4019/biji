
`less` 命令是一个功能强大的终端文本查看器，特别适用于查看大型文件。

## 常用选项

- `-N`：显示每一行的行号。
- `-S`：禁止折行显示，长行将水平滚动而不是换行显示。
- `+<行号>`：从指定的行号开始显示文件内容。

## 常用命令

- `↓` 或 `空格`：向下滚动一页。
- `↑`：向上滚动一页。
- `→`：向右移动。
- `←`：向左移动。
- `j`：向下滚动一行。
- `k`：向上滚动一行。
- `g`：跳到文件开头。
- `G`：跳到文件末尾。
- `/关键词`：搜索匹配的关键词。

## 示例

1. 显示文件内容并显示行号：
   ```bash
less -NS file.txt
   ```

2. 搜索关键词并高亮显示匹配的结果：
   ```bash
less -i file.txt
   ```

以上是 `less` 命令的基本用法及常用选项，希望对您有所帮助。
