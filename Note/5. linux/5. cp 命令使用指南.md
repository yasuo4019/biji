
`cp` 命令用于在 Linux/Unix 系统中复制文件或目录。它可以将一个文件或目录复制到另一个位置，并且具有许多选项以满足不同的需求。

## 命令格式
```bash
cp [选项] 源文件 目标文件
```

## 常用选项

- `-i` 或 `--interactive`：如果目标文件已存在，提示是否覆盖。
- `-r`、`-R` 或 `--recursive`：递归复制目录及其内容。
- `-v` 或 `--verbose`：显示详细的复制过程信息。
- `-p`：保留源文件的属性，包括权限、所有者、组等。
- `-f`：强制复制，即使目标文件已存在也覆盖。

## 示例

1. **复制文件**：
   ```bash
cp source_file destination_file
   ```
   例如：
   ```bash
cp test.fa SXB/Linux/Test/test2.fa
cp gene.list test2.txt
   ```
   这将复制名为 "test.fa" 的源文件，并将其粘贴为名为 "test2.fa" 的目标文件（也可以不写文件名字，直接采用原先名字）。

2. **复制目录及其内容**：
   ```bash
cp -r source_directory destination_directory
   ```
   例如：
   ```bash
cp -r dir1 SXB/Linux/Test/dir2
   ```
   这将递归地复制名为 "dir1" 的源目录及其内容，并将其粘贴为名为 "dir2" 的目标目录。

3. **复制多个文件到目标目录**：
   ```bash
cp file1.txt file2.txt destination_directory
   ```
   例如：
   ```bash
cp test.fa test.gff3 SXB/Linux/Test/
   ```
   这将同时复制 "test.fa" 和 "test.gff3" 两个源文件到目标目录中。
