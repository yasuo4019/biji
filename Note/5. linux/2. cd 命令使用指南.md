 
`cd` 命令是 Linux 和 Unix 系统中的一个常用命令，用于切换当前工作目录。它是 "Change Directory" 的缩写。

## 命令格式
```bash
cd [目录路径]
```

## 示例

1. **切换到指定的目录**：
   ```bash
cd /path/to/directory
   ```
   这将切换到指定的目录路径 `/path/to/directory`。

2. **切换到上级目录**：
   ```bash
cd ..
   ```
   这将切换到当前目录的上级目录。

3. **切换到上次所在的目录**：
   ```bash
cd -
   ```
   这将切换到上次访问的目录。

4. **使用相对路径切换到下级目录**：
   ```bash
cd subdirectory
   ```
   这将切换到当前目录下的 `subdirectory` 目录。

5. **使用通配符进行模糊匹配**：
   ```bash
cd dir*
   ```
   这将切换到当前目录下与 `dir*` 匹配的第一个目录。

6. **返回到当前用户的主目录**：
   ```bash
cd
   ```
   或
   ```bash
cd ~
   ```
   这将切换到当前用户的主目录。

## 注意事项

- 如果目录路径包含空格或特殊字符，应将其用引号括起来。例如：
  ```bash
cd "My Documents/Important Files"
  ```

- `cd` 命令是 Linux 中非常基本和有用的命令，可以帮助你在不同的目录之间切换。
