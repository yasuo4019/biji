
`paste` 命令用于将两个或多个文件的内容合并在一起，以制表符分隔。它可以按列将文件内容交错在一起，方便比较或查看。

## 命令格式
```bash
paste [选项] 文件名
```

## 默认操作
如果没有指定选项，`paste` 命令会将多个文件的行逐个合并在一起。

## 示例

1. **按列合并文件内容**：
   ```bash
paste sample1.exp.bed sample2.exp.bed > combine.list
   ```

2. **使用自定义分隔符**：
   使用 `-d` 选项指定自定义合并后的分隔符。
   ```bash
paste -d "," sample1.exp.bed sample2.exp.bed > combine.list2
   ```

3. **横向合并文件内容**：
   使用 `-s` 选项将文件内容逐行合并为一行（一个文件一行）。
   ```bash
paste -s sample1.exp.bed sample2.exp.bed
   ```
