磁盘管理在Linux和Unix系统中是一个关键的管理任务，通过适当的命令可以监控、管理磁盘的使用情况、配置文件系统等。以下是常用的磁盘管理命令及其功能介绍：

### **1. `df`**
   - **功能**：`df` 命令用于显示文件系统的磁盘空间使用情况，包括每个挂载点的总容量、已用空间、可用空间以及使用率。
   - **基本语法**：
     ```bash
df [选项]
     ```
   - **常用选项**：
     - `-h`：以人类可读的格式显示（例如，使用KB、MB、GB等单位）。
     - `-T`：显示文件系统类型。
     - `-i`：显示inode使用情况，而不是磁盘块使用情况。
   - **示例**：
     ```bash
df -h
     ```
     - **解释**：以人类可读的格式显示磁盘空间使用情况。

     ```bash
df -Th
     ```
     - **解释**：显示每个文件系统的类型和磁盘空间使用情况，以人类可读的格式。

### **2. `du`**
   - **功能**：`du` 命令用于估算文件和目录的磁盘使用量，显示每个文件或目录所占用的磁盘空间。
   - **基本语法**：
     ```bash
du [选项] [目录/文件]
     ```
   - **常用选项**：
     - `-h`：以人类可读的格式显示磁盘使用量。
     - `-s`：只显示总计，不显示每个文件或目录的使用量。
     - `-a`：显示所有文件和目录的使用量，而不仅仅是目录。
   - **示例**：
     ```bash
du -h /home/user
     ```
     - **解释**：显示 `/home/user` 目录下所有文件和子目录的磁盘使用量，以人类可读的格式。

     ```bash
du -sh /var/log
     ```
     - **解释**：仅显示 `/var/log` 目录的总磁盘使用量。

### **3. `fdisk`**
   - **功能**：`fdisk` 是一个用于管理磁盘分区的命令，可以创建、删除、修改分区，以及查看分区表信息。适用于MBR（主引导记录）分区表类型的磁盘。
   - **基本语法**：
     ```bash
fdisk [选项] [设备]
     ```
   - **常用操作**：
     - **启动 `fdisk` 会话**：
       ```bash
fdisk /dev/sda
       ```
     - **常用命令**：
       - `m`：显示帮助菜单。
       - `p`：打印当前分区表。
       - `n`：添加一个新分区。
       - `d`：删除一个现有分区。
       - `w`：将分区表写入磁盘并退出。
       - `q`：退出不保存更改。
   - **注意**：修改分区表后可能需要重新启动系统才能生效。

### **4. `mount` 和 `umount`**
   - **功能**：
     - **`mount`**：将文件系统挂载到指定的目录，使得用户可以访问设备或分区上的数据。
     - **`umount`**：卸载文件系统，使挂载的设备或分区不再可访问。
   - **基本语法**：
     ```bash
mount [选项] 设备 目录
umount [选项] 设备/目录
     ```
   - **常用选项**：
     - `-t`：指定文件系统类型（如 `ext4`、`ntfs`、`vfat` 等）。
     - `-o`：指定挂载选项（如 `ro` 只读，`rw` 读写，`noatime` 禁止更新访问时间等）。
   - **示例**：
     - **挂载一个分区**：
       ```bash
mount /dev/sdb1 /mnt/usb
       ```
       - **解释**：将 `/dev/sdb1` 分区挂载到 `/mnt/usb` 目录。

     - **挂载一个ISO文件**：
       ```bash
mount -o loop file.iso /mnt/iso
       ```
       - **解释**：将ISO文件 `file.iso` 作为文件系统挂载到 `/mnt/iso` 目录。

     - **卸载一个文件系统**：
       ```bash
umount /mnt/usb
       ```
       - **解释**：卸载 `/mnt/usb` 目录中挂载的文件系统。

### **总结**
- **`df`** 和 **`du`** 是查看磁盘使用情况的重要工具，`df` 用于显示文件系统的整体磁盘使用情况，`du` 用于估算文件和目录的具体使用量。
- **`fdisk`** 是一个强大的磁盘分区管理工具，适用于传统的MBR分区表的磁盘管理。
- **`mount`** 和 **`umount`** 命令用于挂载和卸载文件系统，是管理和使用外部设备、分区的重要工具。
