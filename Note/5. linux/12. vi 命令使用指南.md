
`vi` 是一个在 Linux/Unix 系统中常用的文本编辑器，具有丰富的功能和强大的定制能力。

## 命令格式
```bash
vi [选项] 文件名
```

## 常用选项

- `+n`：打开文件并将光标定位在第 n 行处。
- `-d`：比较两个文件。

## 常用命令

- `dd`：删除整行。
- `yy`：复制整行。
- `u`：撤销上一步操作。
- `:w`：保存文件。
- `:q`：退出编辑器。
- `:wq`：保存文件并退出编辑器。
- `:q!`：强制退出编辑器，不保存文件。

## 示例

1. 打开文件：
   ```bash
vi filename
   ```
   例如：
   ```bash
vi peak.xls
   ```
   这将使用 `vi` 编辑器打开名为 "peak.xls" 的文件。

2. 在文件中进行编辑：
   打开文件后，可以使用各种命令来进行编辑。例如，在普通模式下按下 `i` 键进入插入模式，然后输入文本以进行编辑。完成编辑后，按下 `Esc` 键返回普通模式；输入 `:q` 命令退出编辑器；输入 `:wq` 命令保存文件并退出编辑器。
