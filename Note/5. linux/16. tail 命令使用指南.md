
`tail` 命令用于显示文件的末尾部分，默认情况下显示文件的最后10行。

## 常用选项

- `-n num` 或 `--lines=num`：显示文件的最后 num 行，默认为 10 行。
- `-c num` 或 `--bytes=num`：显示文件的最后 num 个字节。
- `-f` 或 `--follow`：以实时追踪模式输出文件的内容，当文件更新时会自动刷新。
- `-F` 或 `--follow=name`：类似 `-f`，但会在文件被删除和重新创建时尝试打开新文件。

## 示例

1. 显示文件的最后 5 行：
   ```bash
tail -n 5 file.txt
   ```

2. 显示文件的最后 20 个字节：
   ```bash
tail -c 20 file.txt
   ```

3. 实时追踪文件内容的更新：
   ```bash
tail -f file.txt
   ```
