### 基本类型

| **关键字**   | **说明** | 取值                                  |
| --------- | ------ | ----------------------------------- |
| `boolean` | 布尔型    | true 或者 false                       |
| `byte`    | 字节型    | 范围是 -128 到 127                      |
| `char`    | 字符型    | 范围从 `\u0000` (0) 到 `\uffff` (65535) |
| `double`  | 双精度浮点  | 双精度的 64 位 IEEE 754 浮点数              |
| `float`   | 单精度浮点  | 单精度的 32 位 IEEE 754 浮点数              |
| `int`     | 整型     | 范围是 -2^31 到 2^31-1                  |
| `long`    | 长整型    | 范围是 -2^63 到 2^63-1                  |
| `short`   | 短整型    | 范围是 -32,768 到 32,767                |

这些关键字是 Java 中的基本数据类型，用于声明不同类型的变量。以下是每个关键字的解释及其典型使用场景：

### `boolean`
- **解释**: `boolean` 类型表示一个布尔值，只有两个可能的取值：`true` 和 `false`。
- **使用场景**: 常用于条件判断和控制流，如 `if` 语句和循环条件。

  **示例**:
  ```java
  boolean isLoggedIn = true;

  if (isLoggedIn) {
      System.out.println("User is logged in.");
  }
  ```

### `byte`
- **解释**: `byte` 类型表示一个 8 位的有符号整数，范围是 -128 到 127。
- **使用场景**: 当需要节省内存或处理二进制数据（如文件 I/O、网络通信）时使用。它也用于在较小的整数范围内存储数据。

  **示例**:
  ```java
  byte smallNumber = 100;
  ```

### `char`
- **解释**: `char` 类型表示一个 16 位的 Unicode 字符，范围从 `\u0000` (0) 到 `\uffff` (65535)。
- **使用场景**: 用于存储单个字符，如字母、数字或符号。常用于处理字符数据或构建字符串。

  **示例**:
  ```java
  char letter = 'A';
  ```

### `double`
- **解释**: `double` 类型表示一个双精度的 64 位 IEEE 754 浮点数。
- **使用场景**: 用于需要高精度的浮点运算，如科学计算、财务应用等。

  **示例**:
  ```java
  double pi = 3.14159;
  ```

### `float`
- **解释**: `float` 类型表示一个单精度的 32 位 IEEE 754 浮点数。
- **使用场景**: 用于需要存储浮点数但对精度要求不高的场合，或者在大规模计算中为了节省内存。

  **示例**:
  ```java
  float temperature = 36.6f; // 'f' 后缀表示这是一个 float 类型的字面量
  ```

### `int`
- **解释**: `int` 类型表示一个 32 位的有符号整数，范围是 -2^31 到 2^31-1。
- **使用场景**: 最常用的整数类型，适用于大多数场景，如计数器、循环变量、计算等。

  **示例**:
  ```java
  int counter = 0;
  ```

### `long`
- **解释**: `long` 类型表示一个 64 位的有符号整数，范围是 -2^63 到 2^63-1。
- **使用场景**: 用于需要存储非常大的整数的场合，如处理大型计算、唯一 ID（如时间戳）、序列号等。

  **示例**:
  ```java
  long distanceToSun = 149_600_000_000L; // 'L' 后缀表示这是一个 long 类型的字面量
  ```

### `short`
- **解释**: `short` 类型表示一个 16 位的有符号整数，范围是 -32,768 到 32,767。
- **使用场景**: 类似于 `byte`，用于需要节省内存且数值范围较小的场合，但比 `byte` 存储更大的数值范围。

  **示例**:
  ```java
  short height = 180;
  ```

### 总结

- **`boolean`**: 用于布尔逻辑操作。
- **`byte`**: 用于处理二进制数据或节省内存的整数操作。
- **`char`**: 用于存储单个字符。
- **`double`**: 用于高精度浮点计算。
- **`float`**: 用于普通精度浮点计算。
- **`int`**: 常用的整数类型。
- **`long`**: 用于处理超大整数。
- **`short`**: 用于较小范围整数并节省内存。
