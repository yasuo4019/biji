## SQL BETWEEN 运算符

### 概述

`BETWEEN` 运算符用于在 `WHERE` 子句中筛选范围内的值。它可以用于数字、日期和字符串类型的数据。

### 基本语法

```sql
SELECT column_name(s)
FROM table_name
WHERE column_name BETWEEN value1 AND value2;
```

### 示例

#### 1. 数字范围

查找价格在 10 到 20 之间的产品：

```sql
SELECT * FROM Products
WHERE Price BETWEEN 10 AND 20;
```

此查询返回所有价格在 10 到 20 之间（包括 10 和 20）的产品。

#### 2. 日期范围

查找在特定日期范围内的订单：

```sql
SELECT * FROM Orders
WHERE OrderDate BETWEEN '2023-01-01' AND '2023-12-31';
```

这条查询返回所有在 2023 年内下的订单。

#### 3. 字符串范围

使用 `BETWEEN` 也可以筛选字符串范围：

```sql
SELECT * FROM Customers
WHERE CustomerName BETWEEN 'A' AND 'M';
```

此查询返回所有客户名字以字母 A 到 M 开头的记录。

### 注意事项

- **包含边界值**：`BETWEEN` 运算符包括指定的边界值。
- **数据类型**：确保 `value1` 和 `value2` 的数据类型与列的数据类型相符。
- **顺序**：`value1` 应小于或等于 `value2`，否则查询将返回空结果。

## SQL NOT BETWEEN 运算符

### 概述

`NOT BETWEEN` 运算符用于筛选不在指定范围内的值。与 `BETWEEN` 相对，它可以有效地查找超出特定范围的数据。

### 基本语法

```sql
SELECT column_name(s)
FROM table_name
WHERE column_name NOT BETWEEN value1 AND value2;
```

### 示例

#### 1. 查找价格不在范围内的产品

要显示价格不在 10 到 20 之间的产品，可以使用 `NOT BETWEEN`：

```sql
SELECT * FROM Products
WHERE Price NOT BETWEEN 10 AND 20;
```

此查询返回所有价格不在 10 到 20 之间的产品。

### 结合 BETWEEN 和 IN

可以将 `BETWEEN` 与 `IN` 运算符结合使用，以进一步筛选数据。

#### 示例

查找价格在 10 到 20 之间且 `CategoryID` 为 1、2 或 3 的产品：

```sql
SELECT * FROM Products
WHERE Price BETWEEN 10 AND 20
AND CategoryID IN (1, 2, 3);
```

此查询返回符合价格和类别条件的产品。

### BETWEEN 文本值

`BETWEEN` 运算符也可以用于字符串，以按字母顺序筛选。

#### 示例

查找产品名称在 "Carnarvon Tigers" 和 "Mozzarella di Giovanni" 之间的所有产品：

```sql
SELECT * FROM Products
WHERE ProductName BETWEEN 'Carnarvon Tigers' AND 'Mozzarella di Giovanni'
ORDER BY ProductName;
```

### NOT BETWEEN 文本值

同样，可以使用 `NOT BETWEEN` 筛选不在特定字母范围内的产品名称。

#### 示例

查找产品名称不在 "Carnarvon Tigers" 和 "Mozzarella di Giovanni" 之间的产品：

```sql
SELECT * FROM Products
WHERE ProductName NOT BETWEEN 'Carnarvon Tigers' AND 'Mozzarella di Giovanni'
ORDER BY ProductName;
```

### BETWEEN 日期

`BETWEEN` 运算符同样适用于日期类型的数据。

#### 示例

查找订单日期在 1996 年 7 月 1 日到 7 月 31 日之间的所有订单：

```sql
SELECT * FROM Orders
WHERE OrderDate BETWEEN '1996-07-01' AND '1996-07-31';
```

### 总结

- **`NOT BETWEEN` 运算符**：用于筛选不在指定范围内的值。
- **结合使用**：可以与 `IN` 运算符结合，进行多条件筛选。
- **文本和日期支持**：适用于字符串和日期数据类型。
- **灵活性**：在数据查询中提供了更多的选择和灵活性。
