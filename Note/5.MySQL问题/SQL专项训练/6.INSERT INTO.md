`INSERT INTO` 语句用于向 SQL 数据库表中插入新记录。它是数据操作语言（DML）的一部分，允许用户添加一行或多行数据。

### 基本语法

#### 1. 插入单行记录

```sql
INSERT INTO table_name (column1, column2, column3)
VALUES (value1, value2, value3);
```

**示例**:
```sql
INSERT INTO employees (name, age, department)
VALUES ('Alice', 30, 'Sales');
```
这个查询向 `employees` 表中插入一名名为 Alice、年龄为 30、部门为 Sales 的员工。

#### 2. 插入多行记录

可以一次插入多条记录，使用逗号分隔每一组 `VALUES`。

```sql
INSERT INTO table_name (column1, column2)
VALUES (value1a, value2a),
       (value1b, value2b),
       (value1c, value2c);
```

**示例**:
```sql
INSERT INTO employees (name, age, department)
VALUES ('Bob', 25, 'HR'),
       ('Charlie', 28, 'IT');
```
这个查询向 `employees` 表中插入两名员工 Bob 和 Charlie。

#### 3. 插入从其他表中选择的记录

可以使用 `INSERT INTO ... SELECT` 语句将数据从一个表插入到另一个表。

```sql
INSERT INTO target_table (column1, column2)
SELECT column1, column2
FROM source_table
WHERE condition;
```

**示例**:
```sql
INSERT INTO employees_archive (name, age, department)
SELECT name, age, department
FROM employees
WHERE age > 50;
```
这个查询将所有年龄大于 50 岁的员工信息插入到 `employees_archive` 表中。

### 注意事项

- **列顺序**: 插入数据时，`VALUES` 中的值顺序应与列顺序一致。
- **默认值**: 如果某列有默认值，可以在 `INSERT` 语句中省略该列。
- **完整性约束**: 插入的数据必须符合表的完整性约束，如主键、外键和非空约束。

### 总结

`INSERT INTO` 语句是将新数据添加到数据库表的基本方法，支持单行和多行插入，并可以从其他表选择数据。
