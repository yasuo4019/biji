在 SQL 中，`MIN()` 和 `MAX()` 函数用于获取特定列中的最小值和最大值。它们通常用于数值、日期或字符串类型的数据。

### 1. `MIN()` 函数

- **功能**: 返回指定列中的最小值。
- **语法**:
  ```sql
  SELECT MIN(column_name) FROM table_name;
  ```

#### 示例

假设有一个名为 `employees` 的表，包含字段 `age`。

```sql
SELECT MIN(age) FROM employees;
```
这个查询返回 `employees` 表中员工的最小年龄。

### 2. `MAX()` 函数

- **功能**: 返回指定列中的最大值。
- **语法**:
  ```sql
  SELECT MAX(column_name) FROM table_name;
  ```

#### 示例

同样在 `employees` 表中，可以使用以下查询：

```sql
SELECT MAX(salary) FROM employees;
```
这个查询返回 `employees` 表中员工的最高薪资。

### 组合使用

这两个函数通常可以与 `GROUP BY` 子句结合使用，以便对分组后的数据进行分析。

#### 示例

```sql
SELECT department, MIN(age) AS youngest, MAX(age) AS oldest
FROM employees
GROUP BY department;
```
这个查询返回每个部门中员工的最小年龄和最大年龄。


### 设置列名（别名）

当使用聚合函数（如 `MIN()` 或 `MAX()`）时，可以使用 `AS` 关键字为返回的列指定描述性名称。

#### 示例
```sql
SELECT MIN(Price) AS SmallestPrice
FROM Products;
```
这个查询返回 `Products` 表中最低价格，并将结果列命名为 `SmallestPrice`。

### 与 `GROUP BY` 结合使用

您还可以将 `MIN()` 函数与 `GROUP BY` 子句结合使用，以获取每个分组的最小值。

#### 示例
```sql
SELECT MIN(Price) AS SmallestPrice, CategoryID
FROM Products
GROUP BY CategoryID;
```
这个查询返回 `Products` 表中每个 `CategoryID` 的最低价格，并将结果列命名为 `SmallestPrice`。

### 总结

- 使用 `AS` 关键字可以为聚合函数的结果指定别名。
- 与 `GROUP BY` 结合使用时，可以计算每个组的最小值，并为结果提供有意义的名称。这样可以提高查询结果的可读性和理解性。
