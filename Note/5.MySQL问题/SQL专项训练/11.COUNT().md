`COUNT()` 是 SQL 中的一个聚合函数，用于计算结果集中的行数或特定列中非 NULL 值的数量。它常用于数据分析和报告中。

### 1. `COUNT()` 的基本用法

#### 计算总行数

使用 `COUNT(*)` 可以计算表中的总行数，包含所有行。

**示例**:
```sql
SELECT COUNT(*) AS TotalEmployees
FROM employees;
```
这个查询返回 `employees` 表中的总行数，并将结果列命名为 `TotalEmployees`。

#### 计算非 NULL 值的数量

使用 `COUNT(column_name)` 计算指定列中非 NULL 值的数量。

**示例**:
```sql
SELECT COUNT(salary) AS NonNullSalaries
FROM employees;
```
这个查询返回 `employees` 表中 `salary` 列非 NULL 值的数量，并将结果列命名为 `NonNullSalaries`。

### 2. 与 `GROUP BY` 结合使用

当与 `GROUP BY` 子句结合使用时，可以计算每个组中的行数。

#### 示例

```sql
SELECT department, COUNT(*) AS EmployeeCount
FROM employees
GROUP BY department;
```
这个查询返回每个部门的员工数量，并将结果列命名为 `EmployeeCount`。

### 3. 使用条件统计

可以使用 `COUNT()` 与 `CASE` 语句结合，计算满足特定条件的行数。

#### 示例

```sql
SELECT 
    COUNT(CASE WHEN salary > 50000 THEN 1 END) AS HighSalaryCount
FROM employees;
```
这个查询返回薪资大于 50,000 的员工数量，并将结果列命名为 `HighSalaryCount`。

### 4. 添加 `WHERE` 子句

可以使用 `WHERE` 子句来指定条件，从而计算满足特定条件的行数。

#### 示例
```sql
SELECT COUNT(ProductID)
FROM Products
WHERE Price > 20;
```
这个查询返回 `Products` 表中价格大于 20 的产品数量。

### 5. 忽略重复项

使用 `COUNT(DISTINCT column_name)` 可以计算指定列中唯一值的数量。

#### 示例
```sql
SELECT COUNT(DISTINCT Price)
FROM Products;
```
这个查询返回 `Products` 表中不同价格的数量。

### 6. 使用别名

可以使用 `AS` 关键字为计数结果指定别名，以提高可读性。

#### 示例
```sql
SELECT COUNT(*) AS [Number of records]
FROM Products;
```
这个查询返回 `Products` 表中的总记录数，并将结果列命名为 `Number of records`。

### 7. 将 `COUNT()` 与 `GROUP BY` 结合使用

结合使用 `COUNT()` 和 `GROUP BY` 可以统计每个分组中的记录数。

#### 示例
```sql
SELECT COUNT(*) AS [Number of records], CategoryID
FROM Products
GROUP BY CategoryID;
```
这个查询返回 `Products` 表中每个 `CategoryID` 的记录数量，并将结果列命名为 `Number of records`。
