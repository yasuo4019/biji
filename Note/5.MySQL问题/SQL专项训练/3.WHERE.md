在 SQL 中，`WHERE` 子句用于指定查询中要筛选的条件。它能够帮助你从数据库中提取符合特定条件的记录。

### 主要功能

- **筛选数据**: `WHERE` 子句用于限制查询返回的行数，只返回满足条件的记录。
- **条件表达式**: 可以使用各种操作符（如 `=`, `!=`, `<`, `>`, `LIKE`, `IN`, `BETWEEN` 等）来构建条件。

### 基本语法

```sql
SELECT column1, column2
FROM table_name
WHERE condition;
```

### 示例

假设有一个名为 `employees` 的表，包含以下字段：`id`, `name`, `age`, `department`。

1. **简单条件**:
   ```sql
   SELECT * FROM employees WHERE age > 30;
   ```
   这个查询返回所有年龄大于 30 的员工。

2. **多条件组合**:
   ```sql
   SELECT * FROM employees WHERE age > 30 AND department = 'Sales';
   ```
   这个查询返回年龄大于 30 且部门为 Sales 的员工。

3. **使用 `LIKE`**:
   ```sql
   SELECT * FROM employees WHERE name LIKE 'A%';
   ```
   这个查询返回所有名字以 "A" 开头的员工。

4. **使用 `IN`**:
   ```sql
   SELECT * FROM employees WHERE department IN ('Sales', 'HR');
   ```
   这个查询返回部门为 Sales 或 HR 的员工。

5. **使用 `BETWEEN`**:
   ```sql
   SELECT * FROM employees WHERE age BETWEEN 25 AND 35;
   ```
   这个查询返回年龄在 25 到 35 之间的员工。

### 总结

- `WHERE` 子句是 SQL 查询中不可或缺的一部分，用于精确控制返回的数据。
- 通过组合多个条件，可以实现复杂的筛选逻辑。
