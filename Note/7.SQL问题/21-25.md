## 21. 什么是索引？

索引是一种数据库结构，用于加速数据检索。它类似于书籍的目录，可以帮助数据库系统快速定位到包含特定值的行，而无需扫描整个表。索引可以显著提高查询性能，尤其是在处理大型表时。

## 22. 你知道哪些类型的索引？

* **唯一索引 (Unique Index)：** 唯一索引确保索引列中不包含重复值。它可以是单列索引，也可以是多列索引。主键约束会自动创建唯一索引。
* **聚簇索引 (Clustered Index)：** 聚簇索引决定了表中数据的物理存储顺序。一个表只能有一个聚簇索引。聚簇索引类似于书籍的页码，它会根据索引键的值对数据进行排序和存储。
* **非聚簇索引 (Non-Clustered Index)：** 非聚簇索引不决定数据的物理存储顺序。它存储指向数据行的指针，类似于书籍目录中的页码。一个表可以有多个非聚簇索引。

## 23. 什么是 Schema？

在数据库管理系统 (DBMS) 中，Schema（模式）是指数据库的逻辑结构和组织方式。它定义了数据库中包含哪些对象（例如表、视图、存储过程、函数、触发器等），以及这些对象之间的关系。

Schema 还定义了数据的类型、格式、约束以及访问权限等信息。简单来说，Schema 就是数据库的设计蓝图。

## 24. 什么是 SQL 注释？

SQL 注释是嵌入在 SQL 代码中的文本，用于解释代码的功能、目的或其他相关信息。数据库管理系统在执行 SQL 代码时会忽略注释。

SQL 注释有两种类型：

* **单行注释：** 以两个连续的减号 (--) 开始，到行尾结束。
* **多行注释：** 以 /* 开始，以 */ 结束，可以跨越多行。

## 25. 什么是 SQL 运算符？

SQL 运算符是用于执行特定操作的符号或关键字。它们通常与 `WHERE` 子句一起使用，用于指定数据筛选条件。

SQL 运算符主要分为以下几类：

* **比较运算符：** 用于比较两个值，例如 `=`（等于）、`>`（大于）、`<`（小于）、`>=`（大于等于）、`<=`（小于等于）、`<>` 或 `!=`（不等于）。
* **逻辑运算符：** 用于组合多个条件，例如 `AND`（与）、`OR`（或）、`NOT`（非）。
* **算术运算符：** 用于执行数学运算，例如 `+`（加）、`-`（减）、`*`（乘）、`/`（除）、`%`（取模）。
* **其他运算符：**  例如 `LIKE`（模式匹配）、`BETWEEN`（范围查询）、`IN`（集合包含）、`IS NULL`（空值判断）等。
