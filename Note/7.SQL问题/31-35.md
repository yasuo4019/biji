## 31. 如何更新表？

使用 `UPDATE` 语句可以修改表中现有数据。

**语法：**

```sql
UPDATE table_name
SET column1 = value1, column2 = value2, ...
WHERE condition;
```

**解释：**

* `UPDATE`：用于更新表数据。
* `table_name`：要更新的表名。
* `SET`：指定要更新的列和新值。
* `column1 = value1, column2 = value2, ...`：要更新的列和对应的新值，多个列之间用逗号分隔。
* `WHERE`：可选的过滤条件，用于指定要更新哪些行。如果不指定 `WHERE` 子句，则会更新表中的所有行。

**示例：**

```sql
-- 将 customer_id 为 1 的客户的邮箱地址更新为 new_email@example.com
UPDATE customers
SET customer_email = 'new_email@example.com'
WHERE customer_id = 1;
```

## 32. 如何从数据库中删除表？

使用 `DROP TABLE` 语句可以从数据库中永久删除表。

**语法：**

```sql
DROP TABLE table_name;
```

**解释：**

* `DROP TABLE`：用于删除表。
* `table_name`：要删除的表名。

**注意：**

* `DROP TABLE` 语句会永久删除表及其所有数据，操作不可恢复。
* 如果要删除表结构但保留数据，可以使用 `TRUNCATE TABLE` 语句。

## 33. 如何获取表中的记录数？

使用聚合函数 `COUNT(*)` 可以获取表中的总行数。

**语法：**

```sql
SELECT COUNT(*) FROM table_name;
```

**解释：**

* `SELECT`：用于检索数据。
* `COUNT(*)`：聚合函数，用于统计行数。
* `FROM`：指定要查询的表名。

## 34. 如何排序表中的记录？

使用 `ORDER BY` 子句可以对查询结果进行排序。

**语法：**

```sql
SELECT column1, column2, ...
FROM table_name
ORDER BY column1 [ASC|DESC], column2 [ASC|DESC], ...;
```

**解释：**

* `ORDER BY`：用于指定排序规则。
* `column1, column2, ...`：要排序的列名，多个列之间用逗号分隔。
* `ASC`：可选的排序方式，表示升序排列，默认值。
* `DESC`：可选的排序方式，表示降序排列。

**示例：**

```sql
-- 按 customer_name 升序排列
SELECT * FROM customers ORDER BY customer_name;

-- 按 order_date 降序排列，如果 order_date 相同，则按 order_amount 升序排列
SELECT * FROM orders ORDER BY order_date DESC, order_amount ASC;
```

## 35. 如何选择表中的所有列？

在 `SELECT` 语句中使用星号 `*` 可以选择表中的所有列。

**语法：**

```sql
SELECT * FROM table_name;
```

**解释：**

* `SELECT *`：表示选择所有列。
* `FROM`：指定要查询的表名。

**注意：**

* 虽然使用 `*` 可以方便地选择所有列，但在实际开发中，建议明确指定要查询的列，以提高查询效率和可读性。
