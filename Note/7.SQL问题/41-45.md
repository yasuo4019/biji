## 41. SQL 中什么是函数，为什么要使用函数？

函数是执行特定任务的一组 SQL 语句。函数接收输入参数，对它们执行计算或其他操作，然后返回结果。函数有助于：

* **提高代码可读性**:  将复杂的逻辑封装在函数中，使代码更易于理解和维护。
* **避免重复代码**:  将常用的代码片段定义为函数，可以在需要的地方重复使用，减少代码冗余。
* **模块化开发**:  函数可以将大型任务分解成更小的、可管理的模块，提高代码的可维护性和可扩展性。


## 42. 你知道哪些类型的 SQL 函数？

SQL 函数可以根据其操作的数据类型和返回值类型进行分类：

**1. 聚合函数**: 

* 针对处理分组表中列的记录，并返回单个值（通常按组）。
* 例如，计算一组学生的平均成绩，可以使用 `AVG()` 函数。

**2. 标量函数**: 

* 处理每个单独的值，并返回单个值。
* 例如，要将字符串转换为大写，可以使用 `UCASE()` 函数。

此外，SQL 函数还可以分为：

* **内置函数**: 由数据库管理系统预先定义的函数，可以直接使用。
* **用户定义函数 (UDF)**: 由用户根据其特定需求创建的函数，可以使用 SQL 或其他编程语言编写。


## 43. 你知道哪些聚合函数？

以下是常用的聚合函数：

* `AVG()`: 返回指定列中所有非空值的平均值。
* `SUM()`: 返回指定列中所有非空值的总和。
* `MIN()`: 返回指定列中的最小值。
* `MAX()`: 返回指定列中的最大值。
* `COUNT()`: 返回指定列中非空值的个数。
* `COUNT(*)`: 返回表中的总行数，包括具有空值的行数。
* `FIRST()`: 返回指定列中的第一个值 (某些数据库不支持)。
* `LAST()`: 返回指定列中的最后一个值 (某些数据库不支持)。


## 44. 你知道哪些标量函数？

以下是常用的标量函数：

**字符串函数**:

* `LEN()` (或 `LENGTH()`): 返回字符串的长度，包括空格。
* `UCASE()` (或 `UPPER()`): 将字符串转换为大写。
* `LCASE()` (或 `LOWER()`): 将字符串转换为小写。
* `INITCAP()`: 将字符串的每个单词的首字母转换为大写。
* `MID()` (或 `SUBSTR()`): 从字符串中提取子字符串。

**数值函数**:

* `ROUND()`: 将数值四舍五入到指定的位数。

**日期/时间函数**:

* `NOW()`: 返回当前日期和时间。


## 45. 什么是大小写处理函数？举一些例子。

大小写处理函数是专门用于处理字符串大小写的标量函数，可以将数据转换为大写、小写或首字母大写。 

以下是一些常见的大小写处理函数：

* `UCASE()` (或 `UPPER()`): 将字符串转换为大写。例如，`UCASE('hello')` 返回 'HELLO'。
* `LCASE()` (或 `LOWER()`): 将字符串转换为小写。例如，`LCASE('Hello')` 返回 'hello'。
* `INITCAP()`: 将字符串的每个单词的首字母转换为大写，其他字母转换为小写。例如，`INITCAP('hello world')` 返回 'Hello World'。
