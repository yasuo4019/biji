## 26. 你知道哪些类型的 SQL 运算符？

SQL 运算符用于操作数据并返回结果。常见的 SQL 运算符类型包括：

* **算术运算符：** 用于执行数学运算。
    * `+`：加法
    * `-`：减法
    * `*`：乘法
    * `/`：除法
    * `%`：取模（返回余数）
* **比较运算符：** 用于比较两个表达式。
    * `=`：等于
    * `>`：大于
    * `<`：小于
    * `>=`：大于等于
    * `<=`：小于等于
    * `<>` 或 `!=`：不等于
* **复合运算符：** 用于将算术运算符和赋值运算符组合在一起。
    * `+=`：加等于
    * `-=`：减等于
    * `*=`：乘等于
    * `/=`：除等于
    * `%=`：取模等于
* **逻辑运算符：** 用于组合多个条件。
    * `AND`：与，所有条件都必须为真
    * `OR`：或，至少一个条件为真
    * `NOT`：非，反转条件的真假
* **字符串运算符：** 用于操作字符串。
    * `||` 或 `+`：连接字符串
    * `LIKE`：模式匹配
    * `%`：匹配任意长度的字符序列
    * `_`：匹配单个字符
* **集合运算符：** 用于组合多个查询的结果集。
    * `UNION`：返回两个结果集的所有不重复行
    * `UNION ALL`：返回两个结果集的所有行，包括重复行
    * `INTERSECT`：返回两个结果集的公共行
    * `EXCEPT` 或 `MINUS`：返回第一个结果集中存在但第二个结果集中不存在的行

## 27. 什么是别名？

别名是为表或列指定的临时名称，用于简化 SQL 查询语句并提高可读性。可以使用关键字 `AS` 来定义别名，也可以省略 `AS` 直接在表名或列名后面加上别名。

**示例：**

```sql
-- 为表定义别名
SELECT c.customer_id, c.customer_name, o.order_id, o.order_date
FROM customers c JOIN orders o ON c.customer_id = o.customer_id;

-- 为列定义别名
SELECT customer_id AS cid, customer_name AS cname
FROM customers;
```

## 28. 什么是子句？

子句是 SQL 查询语句的组成部分，用于指定查询条件、操作或限制。常见的 SQL 子句包括：

* `WHERE`：用于过滤数据，指定查询条件。
* `FROM`：用于指定要查询的表或视图。
* `GROUP BY`：用于对查询结果进行分组。
* `HAVING`：用于对分组后的结果进行过滤。
* `ORDER BY`：用于对查询结果进行排序。
* `LIMIT`：用于限制查询结果返回的行数。

## 29. SELECT 查询中常用的语句有哪些？

`SELECT` 查询语句用于从数据库中检索数据。常用的语句和子句包括：

* `SELECT`：指定要检索的列。
* `FROM`：指定要查询的表或视图。
* `WHERE`：用于过滤数据，指定查询条件。
* `GROUP BY`：用于对查询结果进行分组。
* `HAVING`：用于对分组后的结果进行过滤。
* `ORDER BY`：用于对查询结果进行排序。
* `LIMIT`：用于限制查询结果返回的行数。
* `JOIN`：用于将多个表的数据连接在一起。

## 30. 如何创建表？

可以使用 `CREATE TABLE` 语句来创建新表。

**语法：**

```sql
CREATE TABLE table_name (
    column1 datatype [constraints],
    column2 datatype [constraints],
    ...
);
```

**示例：**

```sql
CREATE TABLE customers (
    customer_id INT PRIMARY KEY,
    customer_name VARCHAR(255) NOT NULL,
    customer_email VARCHAR(255) UNIQUE
);
```

**解释：**

* `CREATE TABLE`：用于创建新表。
* `customers`：新表的名称。
* `column1`, `column2`, ...：表中各个列的名称。
* `datatype`：列的数据类型，例如 `INT`、`VARCHAR`、`DATE` 等。
* `constraints`：可选的列约束，例如 `PRIMARY KEY`、`NOT NULL`、`UNIQUE`、`FOREIGN KEY` 等。
