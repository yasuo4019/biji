## 76. 如何在表中插入多行数据？

可以使用 `INSERT INTO` 语句和多组 `VALUES` 子句一次性向表中插入多行数据。

**语法:**

```sql
INSERT INTO table_name (column1, column2, ...) 
VALUES 
    (value1_1, value1_2, ...),
    (value2_1, value2_2, ...),
    (value3_1, value3_2, ...),
    ...;
```

* 每组括号 `()` 表示一行数据。
* 每行数据中的值的数量和顺序必须与列列表一致。
* 多组 `VALUES` 子句之间使用逗号 `,` 分隔。

**示例:**

```sql
-- 向 employees 表中插入三条记录
INSERT INTO employees (employee_id, first_name, last_name, department_id) 
VALUES 
    (101, 'Alice', 'Smith', 2),
    (102, 'Bob', 'Johnson', 1),
    (103, 'Charlie', 'Williams', 3);
```


## 77. 如何找到表中某一列的第 N 个最高值？

可以使用 `ORDER BY` 子句对目标列进行降序排序，然后使用 `LIMIT` 和 `OFFSET` 子句获取指定位置的值。

**语法:**

```sql
SELECT column_name
FROM table_name
ORDER BY column_name DESC
LIMIT 1 OFFSET (N-1);
```

* `column_name`: 要查找第 N 个最高值的列名。
* `table_name`: 表名。
* `N`: 要查找的第 N 个最高值。

**示例:**

```sql
-- 查找 employees 表中 salary 列的第 6 个最高值
SELECT salary
FROM employees
ORDER BY salary DESC
LIMIT 1 OFFSET 5;
```

**解释:**

1. `ORDER BY salary DESC`: 按照 `salary` 列降序排序。
2. `LIMIT 1`: 只返回一行数据。
3. `OFFSET 5`: 跳过前 5 行数据，从第 6 行开始返回。


## 78. 如何查找表格文本列中以某个字母开头的值？

可以使用 `LIKE` 运算符和通配符来查找以特定字母开头的值。

* `%`: 匹配任意长度的任意字符（包括空字符串）。
* `_`: 匹配任意单个字符。

**示例:**

* 查找 `customers` 表中所有以字母 "A" 开头的 `last_name`：

```sql
SELECT * FROM customers WHERE last_name LIKE 'A%';
```

* 查找 `customers` 表中所有 `last_name` 以字母 "A" 开头且至少包含两个字母的记录：

```sql
SELECT * FROM customers WHERE last_name LIKE 'A_%'
```


## 79. 如何查找表中最后一个 id？

查找表中最后一个 `id` 的方法取决于数据库系统对自增主键的处理方式。

**对于大多数数据库系统，可以使用以下方法:**

* **假设 `id` 是自增主键:**  在这种情况下，最后一个 `id` 通常是最大值。可以使用 `MAX()` 函数找到最大值。

```sql
SELECT MAX(id) FROM table_name;
```

**对于某些数据库系统，例如 SQL Server，可以使用以下方法:**

* **使用 `ORDER BY` 和 `TOP`:** 可以使用 `ORDER BY` 子句对 `id` 列进行降序排序，然后使用 `TOP` 子句选择第一行。

```sql
SELECT TOP 1 id FROM table_name ORDER BY id DESC;
```


## 80. 如何从表中随机选择行？

可以使用数据库系统提供的随机函数和 `ORDER BY` 子句来随机选择行。

**示例:**

* **MySQL:**

```sql
SELECT * FROM table_name ORDER BY RAND() LIMIT 5;
```

* **PostgreSQL:**

```sql
SELECT * FROM table_name ORDER BY RANDOM() LIMIT 5;
```

**解释:**

1. `ORDER BY RAND() / RANDOM()`:  使用随机函数对结果集进行随机排序。
2. `LIMIT 5`:  限制返回的行数为 5。


**注意:**

* 不同的数据库系统可能使用不同的随机函数，请参考相应的文档。
* 随机排序可能会导致性能问题，尤其是在大型表上。
