给定一个 `n` 个元素有序的（升序）整型数组 `nums` 和一个目标值 `target`  ，写一个函数搜索 `nums` 中的 `target`，如果目标值存在返回下标，否则返回 `-1`。

  
**示例 1:**

**输入:** `nums` = [-1,0,3,5,9,12], `target` = 9
**输出:** 4
**解释:** 9 出现在 `nums` 中并且下标为 4

**示例 2:**

**输入:** `nums` = [-1,0,3,5,9,12], `target` = 2
**输出:** -1
**解释:** 2 不存在 `nums` 中因此返回 -1

**提示：**

1. 你可以假设 `nums` 中的所有元素是不重复的。
2. `n` 将在 `[1, 10000]`之间。
3. `nums` 的每个元素都将在 `[-9999, 9999]`之间。

##### Java解法和解析：

``` java
class Solution {
    public int search(int[] nums, int target) {
        int left = 0, right = nums.length - 1; // 初始化左右边界
        while (left <= right) { // 当搜索范围没有交叉时，继续循环
            int mid = (right - left) / 2 + left; // 计算中间元素的索引
            int num = nums[mid]; // 获取中间元素的值

            if (num == target) { // 找到目标值
                return mid; // 返回目标值的索引
            } else if (num > target) { // 中间元素大于目标值
                right = mid - 1; // 在左侧子数组中继续搜索，更新右边界
            } else { // 中间元素小于目标值
                left = mid + 1; // 在右侧子数组中继续搜索，更新左边界
            }
        }
        return -1; // 未找到目标值，返回 -1
    }
    public static void main(String[] args) {
        Solution solution = new Solution();
        int[] nums = {-1, 0, 3, 5, 9, 12};
        int target = 9;
        int index = solution.search(nums, target);
        System.out.println("Target " + target + " is at index " + index);
    }
}
```

**步骤拆解：**

1. **初始化：**
    
    - `left = 0`： `left` 指针指向数组的第一个元素，表示搜索范围的左边界。
    - `right = nums.length - 1`： `right` 指针指向数组的最后一个元素，表示搜索范围的右边界。
2. **循环搜索：**
    
    - `while (left <= right)`： 只要左边界小于等于右边界，就说明搜索范围还没有缩小到空，继续循环查找。
    - `mid = (right - left) / 2 + left`： 计算中间元素的索引。
        - `(right - left) / 2` 计算出中间元素与左边界 `left` 的距离，加上 `left` 就得到了中间元素在数组中的实际索引。
        - 这种写法是为了避免 `(left + right) / 2` 可能会出现的整数溢出问题。
    - `num = nums[mid]`： 获取中间元素的值。
3. **比较并更新边界：**
    
    - `if (num == target)`： 如果中间元素的值等于目标值，说明找到了目标值，直接返回中间元素的索引 `mid`。
    - `else if (num > target)`： 如果中间元素的值大于目标值，说明目标值只可能出现在中间元素的左侧，所以更新右边界 `right = mid - 1`，缩小搜索范围到左半部分。
    - `else` (即 `num < target`)： 如果中间元素的值小于目标值，说明目标值只可能出现在中间元素的右侧，所以更新左边界 `left = mid + 1`，缩小搜索范围到右半部分。
4. **返回结果：**
    
    - 如果循环结束还没有找到目标值，说明目标值不存在于数组中，返回 -1。

**例子：**

假设 `nums = [-1, 0, 3, 5, 9, 12]`, `target = 9`：

1. `left = 0`, `right = 5`, `mid = 2`, `nums[mid] = 3`， `3 < 9`，更新 `left = 3`。
2. `left = 3`, `right = 5`, `mid = 4`, `nums[mid] = 9`, `9 == 9`，找到目标值，返回索引 `4`。

**总结：**

二分查找算法每次将搜索范围缩小一半，时间复杂度为 O(log n)，效率很高。但是它要求数组必须是有序的，如果数组无序，则需要先进行排序才能使用二分查找。
