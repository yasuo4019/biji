
`ln` 命令是 Linux/Unix 系统中用于创建链接的工具。链接可以简单理解为文件的“快捷方式”，它可以指向文件或目录。 `ln` 命令提供了两种类型的链接：

* **硬链接 (hard link):**  硬链接是文件的另一个入口点，它与原始文件共享 inode 信息（可以理解为文件的身份证号码）。这意味着硬链接和原始文件实际上是同一个文件，只是拥有不同的文件名。删除任何一个链接都不会影响其他链接或原始文件。 
* **符号链接 (symbolic link 或 soft link):** 符号链接是一个独立的文件，它包含指向目标文件或目录的路径信息。访问符号链接时，系统会自动跳转到目标文件或目录。删除符号链接不会影响目标文件，但如果目标文件被删除，符号链接将失效。

**命令格式:**

```
ln [选项] 源文件 目标文件
```

**常用选项:**

* `-s, --symbolic`: 创建符号链接。
* `-f, --force`:  如果目标文件已存在，则强制删除后再创建链接。
* `-i, --interactive`: 如果目标文件已存在，则询问用户是否覆盖。

**例子:**

1. **创建符号链接:**

   ```bash
ln -s source_file symlink
   ```

   这将在当前目录中创建一个名为 "symlink" 的符号链接，指向源文件 "source_file"。

2. **强制创建符号链接，覆盖已存在的目标文件:**

   ```bash
ln -fs source_file target 
   ```

   这将强制删除已存在的目标文件 "target"，并创建一个指向源文件 "source_file" 的符号链接。

3. **创建指向目录的符号链接:**

   ```bash
ln -s /path/to/directory  my_directory
   ```

   这将创建一个名为 "my_directory" 的符号链接，指向 `/path/to/directory` 目录。

**注意:**

* 创建硬链接时，源文件和目标文件必须位于同一个文件系统中。
* 符号链接可以跨文件系统创建。
