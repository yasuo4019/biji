
`sort` 命令在 Linux 和 Unix 系统中用于对文本文件的行进行排序操作。它将文件内容按照指定顺序重新排列，并将结果输出到标准输出。

## 命令格式
```bash
sort [选项] [文件]
```

## 常用选项

- `-d` 或 `--dictionary-order`：按照字典顺序进行排序。
- `-f` 或 `--ignore-case`：忽略大小写进行排序。
- `-g` 或 `--general-numeric-sort`：按照一般数值顺序（支持科学记数法）进行排序。
- `-n` 或 `--numeric-sort`：按照数值顺序进行排序。
- `-r` 或 `--reverse`：反向排序，即逆序。
- `-u` 或 `--unique`：去除重复行，只保留其中一行。
- `-k` 或 `--key=KEYDEF`：按照指定的键进行排序（格式为 `startpos,endpos`，单列时可只写 `startpos`）。
- `-t` 或 `--field-separator=SEP`：设置字段分隔符，用于指定字段排序。

## 示例

1. 对文件行进行字典顺序排序：
   ```bash
sort test.gff3
   ```

2. 按照指定字段进行排序（使用空格作为字段分隔符）：
   ```bash
sort -k4,4n test.gff3
sort -t, -k2,2n exp.csv
   ```

3. 按照字符串顺序逆序排序：
   ```bash
sort -k1,1r peak.xls
   ```
