
`rm` 命令是 Linux/Unix 系统中用于删除文件和目录的工具。它会 **永久性地删除** 文件和目录，因此在使用时需要格外谨慎。

**命令格式:**

```
rm [选项] 文件或目录
```

**常用选项:**

* `-i, --interactive`: 删除前进行交互式确认，询问用户是否确认删除。
* `-r, -R, --recursive`: 递归删除目录及其内容。 
* `-f, --force`: 强制删除，不进行任何提示或确认。

**例子:**

1. **删除文件:**

   ```bash
rm test2.txt
   ```

   这将删除名为 "test2.txt" 的文件。

2. **递归删除目录及其内容:**

   ```bash
rm -r dir1
   ```

   这将递归地删除名为 "dir1" 的目录及其所有内容。

3. **强制删除文件，无需确认:**

   ```bash
rm -f test1
   ```

   这将强制删除名为 "test1" 的文件，不进行任何确认。

4. **交互式地删除文件:**

   ```bash
rm -i file3.txt
   ```

   这将在删除文件 "file3.txt" 之前进行交互式确认，询问用户是否确认删除。

**注意:**

* 使用 `rm` 命令时要格外小心，尤其是使用 `-f` 选项时，因为它会永久性地删除文件或目录，并且无法恢复。
* 如果不确定某个命令的作用，请先使用 `man rm` 命令查看其帮助文档。
