
`cat` 命令用于在终端中查看文件内容，同时还可以用于将多个文件合并输出到一个文件。它是 "concatenate"（连接）的缩写。

## 命令格式
```bash
cat [选项] 文件...
```

## 常用选项

- `-n, --number`：同时显示行号。
- `-T, --show-tabs`：将制表符（Tab）显示为 `^I` 。
- `-A, --show-all`：显示所有特殊字符。

## 示例

1. 查看文件内容：
   ```bash
cat file.txt
   ```
   这将在终端中显示名为 "file.txt" 的文件的内容。

2. 同时显示行号：
   ```bash
cat -n file.txt
   ```
   这将在终端中显示名为 "file.txt" 的文件的内容，并在每行前显示行号。

3. 合并多个文件并输出到一个新文件：
   ```bash
cat file1.txt file2.txt > merged.txt
   ```
   这将名为 "file1.txt" 和 "file2.txt" 的文件内容合并，并将输出重定向到一个名为 "merged.txt" 的新文件中。

4. 查看文件的全部符号（空格和制表符区别）：
   ```bash
cat -A file.txt
   ```

5. 追加多个文件的内容到一个已有文件：
   ```bash
cat file1.txt file2.txt >> existing.txt
   ```
   这将名为 "file1.txt" 和 "file2.txt" 的文件内容追加到名为 "existing.txt" 的已有文件末尾。
