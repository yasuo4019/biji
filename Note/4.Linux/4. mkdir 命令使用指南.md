
`mkdir` 命令用于在 Linux/Unix 系统中创建目录。它允许用户根据需要在指定位置创建一个或多个目录。

## 命令格式
```bash
mkdir [选项] 目录名...
```

## 常用选项

- `-m` 或 `--mode <模式>`：设置创建目录的权限模式。例如，`-m 755` 设置目录权限为 `rwxr-xr-x`。
- `-p` 或 `--parents`：创建嵌套目录。如果上级目录不存在，也会一并创建。
- `-v` 或 `--verbose`：显示详细的创建过程信息。

## 示例

1. **创建单个目录**：
   ```bash
mkdir directory_name
   ```
   例如：
   ```bash
mkdir SXB
   ```
   这将在当前工作目录中创建名为 "SXB" 的目录。

2. **创建多个目录**：
   ```bash
mkdir dir1 dir2 dir3
   ```
   例如：
   ```bash
mkdir dir1 dir2 dir3
   ```
   这将在当前工作目录中同时创建 "dir1"、"dir2" 和 "dir3" 三个目录。

3. **递归创建目录（创建嵌套目录）**：
   ```bash
mkdir -p SXB/Linux/Test
   ```
   这将递归地创建一个包含多个层级的目录结构。它会创建 "SXB" 目录、在其中创建 "Linux" 目录，然后在 "Linux" 目录中创建 "Test" 目录。

4. **设置权限模式并创建目录**：
   ```bash
mkdir -m 755 new_directory
   ```
   这将创建一个名为 "new_directory" 的目录，并将其权限设置为 `755`。
