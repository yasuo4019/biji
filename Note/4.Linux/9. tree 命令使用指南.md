
`tree` 命令用于以树状结构显示指定目录下的文件和子目录。它可以很方便地帮助我们了解目录结构和文件的层次关系。

## 命令格式
```bash
tree [选项] [目录]
```

## 常见选项

- `-a` 或 `--all`：显示所有文件和目录，包括隐藏的。
- `-d` 或 `--dirs-only`：只显示目录，不显示文件。
- `-L <级别>` 或 `--max-depth=<级别>`：限制显示的层数。
- `-f` 或 `--full-path`：显示文件和目录的完整路径。

请注意，`tree` 命令在不同的操作系统或环境中可能存在差异，某些选项可能会有所不同。

## 示例

1. **显示当前目录的树状结构**：
   ```bash
tree
   ```
   默认不显示隐藏文件。

2. **显示指定目录的树状结构**：
   ```bash
tree /path/to/directory
   ```
   例如：
   ```bash
tree scRNA-seq/
   ```

3. **显示指定目录的树状结构，包括隐藏文件**：
   ```bash
tree -a /path/to/directory
   ```

4. **只显示指定目录的子目录**：
   ```bash
tree -d /path/to/directory
   ```
   例如：
   ```bash
tree -d scRNA-seq/
   ```

5. **限制显示层数**：
   ```bash
tree -L 2 /path/to/directory
   ```
   这将显示指定目录的树状结构，最多显示两层级别。
