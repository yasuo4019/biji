
`LEFT JOIN` 关键字用于返回左表（table1）中的所有记录，以及右表（table2）中匹配的记录。如果右表没有匹配的记录，则结果将返回 `NULL`。

### LEFT JOIN 语法

```sql
SELECT column_name(s)
FROM table1
LEFT JOIN table2 ON table1.column_name = table2.column_name;
```

> **注意**: 在某些数据库中，`LEFT JOIN` 也称为 `LEFT OUTER JOIN`。

### LEFT JOIN 示例

以下 SQL 语句将选择所有客户及其可能拥有的订单：

```sql
SELECT Customers.CustomerName, Orders.OrderID
FROM Customers
LEFT JOIN Orders ON Customers.CustomerID = Orders.CustomerID
ORDER BY Customers.CustomerName;
```

### 总结

- **LEFT JOIN 的用途**：返回左表中的所有记录，即使右表没有匹配的记录。
- **结果特征**：左表的每个记录都会显示，右表没有匹配时返回 `NULL`。
- **应用场景**：当需要获取左表所有数据，同时查看右表的相关数据时，`LEFT JOIN` 是非常有用的。

通过 `LEFT JOIN`，可以有效地从两个表中提取信息，同时确保左表的所有记录都在结果中显示。
