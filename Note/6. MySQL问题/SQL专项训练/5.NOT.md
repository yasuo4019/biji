在 SQL 中，`NOT` 运算符用于反转条件的布尔值。它可以用于 `WHERE` 子句中，帮助排除特定的条件或记录。

### 功能

- **反转条件**: `NOT` 运算符将条件的结果取反。如果条件为真，`NOT` 会使其变为假，反之亦然。

### 基本用法

#### 1. 使用 `NOT`

**示例**:
```sql
SELECT * FROM employees
WHERE NOT department = 'Sales';
```
这个查询返回所有部门不是 Sales 的员工。

#### 2. 与其他条件结合使用

`NOT` 可以与 `AND` 和 `OR` 运算符结合使用。

**示例**:
```sql
SELECT * FROM employees
WHERE NOT (age > 30 AND department = 'Sales');
```
这个查询返回所有不满足“年龄大于 30 且部门为 Sales”的员工。

#### 3. 与 `IN` 和 `LIKE` 一起使用

- **使用 `NOT IN`**:
```sql
SELECT * FROM employees
WHERE department NOT IN ('Sales', 'HR');
```
这个查询返回部门不是 Sales 和 HR 的所有员工。

- **使用 `NOT LIKE`**:
```sql
SELECT * FROM employees
WHERE name NOT LIKE 'A%';
```
这个查询返回所有名字不以 "A" 开头的员工。

### 总结

- `NOT` 运算符用于排除符合特定条件的记录。
- 可以与其他逻辑运算符结合使用，以构建复杂的查询条件。
