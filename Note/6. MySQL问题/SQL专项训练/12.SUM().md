`SUM()` 是 SQL 中的一个聚合函数，用于计算指定列的总和。它通常用于数值数据，帮助分析数据集中的总值。

### 1. 基本用法

#### 计算总和

使用 `SUM(column_name)` 可以计算指定列的总和。

#### 示例
```sql
SELECT SUM(salary) AS TotalSalary
FROM employees;
```
这个查询返回 `employees` 表中所有员工薪资的总和，并将结果列命名为 `TotalSalary`。

### 2. 与 `WHERE` 子句结合使用

可以使用 `WHERE` 子句来过滤数据，仅计算符合条件的行的总和。

#### 示例
```sql
SELECT SUM(salary) AS TotalHighSalaries
FROM employees
WHERE salary > 50000;
```
这个查询返回薪资大于 50,000 的员工的总薪资，并将结果列命名为 `TotalHighSalaries`。

### 3. 忽略 NULL 值

`SUM()` 函数会自动忽略 NULL 值，因此在计算时只会考虑非 NULL 的数值。

### 4. 与 `GROUP BY` 结合使用

结合使用 `SUM()` 和 `GROUP BY` 可以计算每个分组的总和。

#### 示例
```sql
SELECT department, SUM(salary) AS TotalSalary
FROM employees
GROUP BY department;
```
这个查询返回每个部门的员工薪资总和，并将结果列命名为 `TotalSalary`。

### 5. 使用别名

可以使用 `AS` 关键字为计算结果指定别名，以提高结果的可读性。

#### 示例
```sql
SELECT SUM(salary) AS [Total Employee Salary]
FROM employees;
```
这个查询返回所有员工薪资的总和，并将结果列命名为 `Total Employee Salary`。

### 6. 使用表达式的 `SUM()`

`SUM()` 函数的参数可以是一个表达式，这样可以在计算总和时进行更复杂的运算。

#### 示例

假设 `OrderDetails` 表中每件产品的价格是 10 美元，可以通过将每个数量乘以 10 来计算总收入。

```sql
SELECT SUM(Quantity * 10) AS TotalRevenue
FROM OrderDetails;
```
这个查询计算了所有订单的总收入，并将结果列命名为 `TotalRevenue`。

### 7. 使用连接查找实际金额

为了获取每种产品的实际价格，可以将 `OrderDetails` 表和 `Products` 表连接起来，以便计算基于实际价格的总收入。

#### 示例

```sql
SELECT SUM(Price * Quantity) AS TotalRevenue
FROM OrderDetails
LEFT JOIN Products ON OrderDetails.ProductID = Products.ProductID;
```
在这个查询中：
- `LEFT JOIN` 用于将 `OrderDetails` 表与 `Products` 表连接，基于 `ProductID` 字段。
- `SUM(Price * Quantity)` 计算所有订单的实际总收入，并将结果列命名为 `TotalRevenue`。
