
`RIGHT JOIN` 关键字用于返回右表（table2）中的所有记录，以及左表（table1）中匹配的记录。如果左表没有匹配的记录，则结果将返回 `NULL`。

### RIGHT JOIN 语法

```sql
SELECT column_name(s)
FROM table1
RIGHT JOIN table2 ON table1.column_name = table2.column_name;
```

> **注意**: 在某些数据库中，`RIGHT JOIN` 也称为 `RIGHT OUTER JOIN`。

### RIGHT JOIN 示例

以下 SQL 语句将返回所有员工及其可能下的订单：

```sql
SELECT Orders.OrderID, Employees.LastName, Employees.FirstName
FROM Orders
RIGHT JOIN Employees ON Orders.EmployeeID = Employees.EmployeeID
ORDER BY Orders.OrderID;
```

|---------|----------|-----------|
| 10309   | Leverling| Janet     |

| NULL    | Fuller   | Andrew    |

### 总结

- **RIGHT JOIN 的用途**：返回右表中的所有记录，即使左表没有匹配的记录。
- **结果特征**：右表的每个记录都会显示，左表没有匹配时返回 `NULL`。
- **应用场景**：当需要获取右表所有数据，同时查看左表的相关数据时，`RIGHT JOIN` 是非常有用的。

通过 `RIGHT JOIN`，可以有效地从两个表中提取信息，同时确保右表的所有记录都在结果中显示。
