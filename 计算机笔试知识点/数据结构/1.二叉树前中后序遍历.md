二叉树的遍历分为先序遍历、中序遍历和后序遍历，具体的遍历顺序如下：

### 1. **先序遍历（Pre-order Traversal）**
   - **遍历顺序**：先访问根结点，然后遍历左子树，最后遍历右子树。
   - **步骤**：
     1. 访问根结点。
     2. 递归遍历左子树。
     3. 递归遍历右子树。
   - **顺序模板**：根 → 左 → 右

   **示例**：
   对于如下二叉树：
   ```
       A
      / \
     B   C
    / \
   D   E
   ```
   先序遍历的结果是：`A B D E C`

### 2. **中序遍历（In-order Traversal）**
   - **遍历顺序**：先遍历左子树，然后访问根结点，最后遍历右子树。
   - **步骤**：
     1. 递归遍历左子树。
     2. 访问根结点。
     3. 递归遍历右子树。
   - **顺序模板**：左 → 根 → 右

   **示例**：
   对于上述二叉树：
   ```
       A
      / \
     B   C
    / \
   D   E
   ```
   中序遍历的结果是：`D B E A C`

### 3. **后序遍历（Post-order Traversal）**
   - **遍历顺序**：先遍历左子树，然后遍历右子树，最后访问根结点。
   - **步骤**：
     1. 递归遍历左子树。
     2. 递归遍历右子树。
     3. 访问根结点。
   - **顺序模板**：左 → 右 → 根

   **示例**：
   对于上述二叉树：
   ```
       A
      / \
     B   C
    / \
   D   E
   ```
   后序遍历的结果是：`D E B C A`

### 总结：
- **先序遍历**顺序：根 → 左 → 右
- **中序遍历**顺序：左 → 根 → 右
- **后序遍历**顺序：左 → 右 → 根

每种遍历方式的顺序不同，适用于不同的场景和需求。例如，先序遍历常用于复制树结构，中序遍历用于输出排序结果，而后序遍历常用于删除树结构。
